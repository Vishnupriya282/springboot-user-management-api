# Spring Boot User Management API

A RESTful backend application built using Spring Boot, MySQL, and JPA.

## Features
- Create user
- Get all users
- Get user by id
- Update user
- Delete user

## Tech Stack
- Java
- Spring Boot
- MySQL
- Spring Data JPA
- Maven

## APIs
POST   /api/users
GET    /api/users
GET    /api/users/{id}
PUT    /api/users/{id}
DELETE /api/users/{id}
